🔑 Настройка Wildberries API

Введите токен для доступа к аналитике и заказам

💡 Для чего нужен токен?
Токен позволяет автоматически получать заказы и аналитику с Wildberries.
Токен начинается с "eyJhbGciOiJFUzI1NiIsImtpZCI6..."

📖 Как получить токен Wildberries:

  1. Перейдите в личный кабинет Wildberries
  2. Нажмите Настройки → Доступ к API
  3. Нажмите "Создать новый ключ"
  4. Выберите разрешения: ✅ Статистика и ✅ Заказы
  5. Скопируйте полученный токен
  6. Вставьте его в поле выше и нажмите "Сохранить"

⚠️ Важно: Токен должен иметь права на Статистику для работы аналитики!
